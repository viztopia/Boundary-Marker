# Boundary Marker

A timeline documenting the evolving definitions of "Human" and "Artificial Intelligence" by various AI models.

## Tech Stack
*   **Framework**: Astro
*   **UI**: React
*   **Styling**: Tailwind CSS v4
*   **Deployment**: GitHub Pages

## Development

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build**
    ```bash
    npm run build
    ```

## Adding New Archives
1.  Create a new markdown file in `src/content/archives/` named `YYYYMMDD.md`.
2.  Follow the multi-model format:
    ```markdown
    Model Name
    
    Q1
    -----
    A1
    -----
    Q2
    -----
    A2
    
    ======
    
    Next Model Name
    ...
    ```
